een simpele 2 player game.
wat je moet doen is simpel player 1 beweegt met de arrow keys, player 2 beweegt met AWSD.
de objectif is simpel je speelt tegen elkaar op een speel vlak en wie het meeste van zijn kleur op het speel vlak kleurt WINT de game!

no limits applied dus je kan zo snel mogenlijk drukken als je wilt ;)
